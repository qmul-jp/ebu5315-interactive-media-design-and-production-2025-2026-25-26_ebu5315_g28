# Project Description
Circle Planet is an interactive multimedia web application designed to help GCSE students learn circle-related geometry topics through engaging animations, games, and quizzes.
## Target Users
- GCSE Mathematics students (ages 14-16)
- Self-learners interested in geometry
- Teachers seeking interactive teaching resources
## Learning Objectives
- Understand core circle theorems through visual demonstrations
- Practice problem-solving through interactive games
- Test knowledge through level-based quizzes
- Receive instant feedback via AI-powered assistance
